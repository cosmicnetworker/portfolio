NETGINEER

Empowering Newly Graduated Engineers to Connect, Collaborate, and Grow

Overview

NETGINEER is a freelancing and networking platform designed for recent engineering graduates. It helps them transition from academia to the professional world by allowing them to offer freelance services and build a strong professional network.

Core Features

  • User profile creation with skills and credentials
  • Project bidding and freelance matching system
  • Messaging and feedback system
  • Portfolio showcase and client testimonials
  • Collaboration tools for project teams

Technologies Used

  • HTML, CSS, Bootstrap for UI
  • JavaScript for interactivity
  • PHP + MySQL for backend and database
  • GitHub for version control
