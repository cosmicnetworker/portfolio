UrbanRoots Davao

Hydroponic Gardening for the Urban Future

Overview

UrbanRoots Davao is a hydroponics-based platform designed to bring sustainable urban gardening to life...

Features

  • Hydroponic starter kits for beginners and pros
  • IoT monitoring system to track plant health
  • Urban gardening community portal
  • Mobile app for guidance and support

Technologies Used

  • HTML, CSS, Bootstrap
  • JavaScript for interactive dashboard
  • IoT sensors for environmental data
  • MySQL + PHP backend
